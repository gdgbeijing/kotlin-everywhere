What's New in Kotlin

详细内容稍后更新。

陈琦

陈琦，携程机票研发部 无线研发总监，负责携程 App 机票预订业务的技术研发和管理工作。

携程 Kotlin Multiplatform 工程实践

介绍：1. 携程机票无线团队的 Kotlin 实践进展。2. Kotlin Native 在内部研发工具方面的运用。3. Kotlin Multiplatform 在携程 App 的线上生产实践过程。4. App 从原生到 Kotlin Multiplatform 的重构迁移思路与验证。

贾彦伟

贾彦伟，多年互联网研发从业者，关注编程、架构、技术管理、培训、开源等领域。Kotlin 中文站与 Ktor 中文站负责人。曾在 JetBrains 北京开发者日上做技术分享。

潜力无限：Kotlin 用于服务端与 WebAssembly

介绍将 Kotlin 用于服务端与 WebAssembly 领域的优势与潜力。

陈龙博

陈龙博，抖音 Android 工程师，开源爱好者，对于新技术的 PR 有高度热情，多次在 GDG 做 Kotlin 和 Android 相关的分享。

Boost Your Productivity with Kotlin / 用 Kotlin 提升你的开发效率

通过一些例子来表达 Kotlin 对开发者的友好和工作效率的提升。代码可读性还有稳定性的例子，面向还未使用 Kotlin 和 Kotlin 的初期使用者起到 PR 和指引的作用。

段建华

段建华，前 Flipboard 工程师，网络 ID 为”技术小黑屋“，长期维护个人的技术博客(droidyue.com)。多次在 Droidcon Beijing, GDG, 开源中国等技术会议做分享。

Be Friends with kotlinc

内容涵盖 如何通过配置编译选项控制 kotlinc 更好的编译输出，开发者如何利用 contract 和注解让 kotlinc 更加 smart，以及关于 kotlinc 的编译速度讨论等方面。

朱涛

朱涛，MOMO Android 工程师，Kotlin 技术爱好者，多年 Kotlin Android 开发经验，多次在 GDG 做 Kotlin 和 Android 技术分享。

Kotlin High-Performance Programming - 避开 Kotlin 的性能陷阱

Kotlin 对比 Java，提供了一些新的语言特性，这让 Kotlin 更安全，更简洁，但这同时也为 Kotlin 带来了一些性能问题。这个主题会介绍：Delegation，Functional Programming，Collections 等特性带来的性能开销，以及如何更好的使用这些特性。与此同时，我们也会探讨一些 Android 开发中的实际案例。

高杰

高杰，Android 工程师，HenCoder 及码上开学成员之一。

Kotlin Coroutine / Kotlin 协程

什么是 Kotlin Coroutine ，Kotlin 编译器是怎样实现 Kotlin Coroutine 的，如何更好的引入 Kotlin Coroutine 代码。

乔禹昂

乔禹昂，OKCoin Android 工程师，Kotlin 早期实践者，Kotlin 中文官网译者，主要负责协程与 Native 部分的翻译，Kotlin 中文社区运营。

Kotlin Functional Programming / Kotlin 函数式编程

在 Kotlin 的语法特性中，函数式编程的概念可以说是最让以往的 Java 开发者感到陌生的特性之一了，本次分享将全面介绍 Kotlin 中的函数式编程概念，并帮助大家理解隐藏在语法之下的实现原理。
